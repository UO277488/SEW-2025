Ayuda y guía rápida del proyecto

Esta página explica brevemente las secciones principales del sitio y cómo utilizarlas.

Inicio

La página principal ofrece un resumen con imágenes y noticias relacionadas con MotoGP. Desde aquí puedes acceder rápidamente a las secciones de Piloto, Circuito, Meteorología, Clasificaciones, Juegos y Ayuda.

Piloto

En la sección Piloto encontrarás la biografía, estadísticas y multimedia sobre el piloto destacado (Pecco Bagnaia). Usa los vídeos y audios embebidos para ampliar la información.

Circuito

La sección Circuito permite cargar un archivo InfoCircuito.html para obtener un resumen del circuito, además de cargar archivos KML para ver el trazado en el mapa y SVG de altimetría para visualizar el perfil.

Meteorología

La sección Meteorología permite consultar predicciones y datos históricos (Open‑Meteo). Introduce la fecha de la carrera o de entrenamientos y pulsa el botón correspondiente para obtener los datos.

Clasificaciones

La sección Clasificaciones muestra el ganador y el podio extraídos del fichero XML local xml/circuitoEsquema.xml. Si el XML está disponible, se renderiza automáticamente la información.

Juegos

En la sección de Juegos encontrarás diferentes juegos relacionados con MotoGP:

  • Juego de memoria con cartas: Un juego de memoria donde deberás encontrar parejas de cartas con logos de los equipos de MotoGP. Las cartas están organizadas en una cuadrícula adaptativa que se ajusta a diferentes tamaños de pantalla.
  • Cronómetro: Interfaz para probar la clase Cronometro. Acceder al cronómetro.

Para acceder a los juegos, utiliza los enlaces proporcionados en la página juegos.html.

Ayuda

Esta sección proporciona información sobre cómo utilizar el proyecto MotoGP Desktop y navegar por sus diferentes funcionalidades. Aquí encontrarás detalles sobre cada sección del menú de navegación.
