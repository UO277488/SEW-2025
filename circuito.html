Información sobre los circuitos

Cargar información del circuito

Selecciona el archivo InfoCircuito.html para cargar la información del circuito:

Aquí aparecerá el resumen básico del circuito tras cargar un archivo InfoCircuito.html. Usa el selector "Cargar archivo HTML" para cargar los datos desde tu equipo. Si quieres, también puedes cargar manualmente los archivos KML o SVG para ver mapas y perfiles.

Cargar gráfico de altimetría

Selecciona el archivo altimetria.svg para visualizar el perfil de altimetría:

Sube aquí un archivo altimetria.svg para visualizar el perfil de altimetría del circuito. El gráfico mostrará la altura en función de la distancia y se actualizará al cargar el archivo.
